# 位运算常用操作

> &
- 清零单元: 全部置 0
- 取指定位: 指定位置 1 ，其余位置 0

> |
- 为一个数据某些位置 1： 对应位置 1

> ^
- 翻转特定位： 翻转对应位置 1 ， 其他置 0
- 与 0 异或 保留原值， 与自身异或， 为0
